Research completed: 2026-02-25

## 1. Dune Analytics
- **URL:** https://dune.com/docs/api/
- **What it provides:** On-chain blockchain analytics, custom SQL queries on Ethereum/Polygon/BSC data, DeFi protocol metrics, NFT sales, token transfers
- **Rate limits:** Free tier: 10 queries/day, community dashboards unlimited view
- **Auth:** API key (free tier available)
- **Category:** Crypto on-chain analytics
- **Notes:** Community-built dashboards, rich DeFi protocol data, MEV analytics

## 2. The Graph
- **URL:** https://thegraph.com/docs/en/querying/querying-the-graph/
- **What it provides:** Decentralized blockchain indexing via GraphQL, real-time Ethereum/L2 data, smart contract event tracking
- **Rate limits:** Free tier: 100,000 queries/month
- **Auth:** API key (free tier available)
- **Category:** Crypto real-time data
- **Notes:** Powers many DeFi protocols, excellent for real-time blockchain queries

## 3. Covalent API
- **URL:** https://www.covalenthq.com/docs/api/
- **What it provides:** Multi-chain blockchain data (100+ chains), wallet balances, historical transactions, token prices, NFT metadata
- **Rate limits:** Free tier: 100,000 credits/month
- **Auth:** API key (free)
- **Category:** Crypto multi-chain analytics
- **Notes:** Single API for 100+ blockchains, excellent for portfolio tracking

## 4. Bitquery
- **URL:** https://bitquery.io/products/streaming
- **What it provides:** Real-time blockchain data streams, DEX trades, smart money tracking, token flows, MEV detection
- **Rate limits:** Free tier: 10,000 points/month
- **Auth:** API key (free tier available)
- **Category:** Crypto real-time feeds
- **Notes:** GraphQL interface, streaming WebSocket support

## 5. Santiment
- **URL:** https://api.santiment.net/
- **What it provides:** Crypto social sentiment, development activity (GitHub commits), network growth, exchange flows, whale transactions
- **Rate limits:** Free tier: limited metrics access
- **Auth:** API key (free tier available)
- **Category:** Crypto alternative data - social & dev activity
- **Notes:** Unique GitHub development metrics, social volume analysis

## 6. OpenSky Network
- **URL:** https://openskynetwork.github.io/opensky-api/
- **What it provides:** Real-time flight tracking data, historical flight paths, aviation traffic patterns
- **Rate limits:** Free tier: 100 credits/day (1 credit per request)
- **Auth:** Optional account for higher limits
- **Category:** Alternative data - aviation/travel demand
- **Notes:** Proxy for economic activity, tourism trends, business travel

## 7. AeroDataBox
- **URL:** https://aerodatabox.com/
- **What it provides:** Airport data, flight schedules, aircraft information, airline statistics
- **Rate limits:** Free tier: 150 requests/month
- **Auth:** API key (free tier via RapidAPI)
- **Category:** Alternative data - aviation
- **Notes:** Structured aviation data, good for travel sector analysis

## 8. OECD Data API
- **URL:** https://data.oecd.org/api/
- **What it provides:** Economic indicators for developed nations (GDP, unemployment, inflation, trade, productivity)
- **Rate limits:** None specified (public data)
- **Auth:** None
- **Category:** Macro economic indicators
- **Notes:** High-quality data for OECD member countries, standardized methodology

## 9. Eurostat API
- **URL:** https://ec.europa.eu/eurostat/web/json-and-unicode-web-services
- **What it provides:** European Union economic statistics, regional data, demographics, trade flows
- **Rate limits:** None specified
- **Auth:** None
- **Category:** Macro economic indicators - EU focus
- **Notes:** Authoritative EU data, granular regional breakdowns

## 10. Bank for International Settlements (BIS) Data
- **URL:** https://data.bis.org/
- **What it provides:** Global banking statistics, central bank policy rates, credit aggregates, exchange rates
- **Rate limits:** None (public data)
- **Auth:** None required for bulk downloads
- **Category:** Macro financial indicators
- **Notes:** Central bank of central banks, systemic risk indicators

## 11. Trading Economics API
- **URL:** https://tradingeconomics.com/api
- **What it provides:** 300,000+ economic indicators, earnings calendars, credit ratings, government bonds
- **Rate limits:** Free tier: 1000 requests/month (historical data only)
- **Auth:** API key (free tier available)
- **Category:** Macro economic data
- **Notes:** Comprehensive global coverage, includes emerging markets

## 12. IEX Cloud
- **URL:** https://iexcloud.io/docs/api/
- **What it provides:** Real-time US stock quotes (IEX exchange), company financials, news, insider transactions
- **Rate limits:** Free tier: 50,000 messages/month
- **Auth:** API key (free tier available)
- **Category:** Stock market real-time data
- **Notes:** US exchange data, better than delayed feeds

## 13. EOD Historical Data
- **URL:** https://eodhistoricaldata.com/financial-apis/
- **What it provides:** End-of-day stock prices worldwide, fundamentals, dividends, splits, insider trades
- **Rate limits:** Free tier: 20 requests/day, 1 year history, limited tickers
- **Auth:** API key (free)
- **Category:** Stock market data
- **Notes:** 70+ exchanges worldwide, good for international coverage

## 14. Tradier
- **URL:** https://documentation.tradier.com/brokerage-api
- **What it provides:** Real-time stock/option quotes, Greeks, order book, market hours, historical options data
- **Rate limits:** Sandbox: unlimited (delayed data), Market data subscription for real-time
- **Auth:** API key (sandbox free)
- **Category:** Options market data
- **Notes:** Excellent options chains and Greeks, rare free options data

## 15. WhaleWisdom API
- **URL:** https://whalewisdom.com/
- **What it provides:** 13F institutional holdings, hedge fund portfolios, insider transactions, ownership changes
- **Rate limits:** Free tier: limited web scraping, paid API available
- **Auth:** Account required for structured data
- **Category:** Institutional investor data
- **Notes:** Structured access to SEC 13F filings, tracks superinvestors

## 16. Dataroma
- **URL:** https://www.dataroma.com/m/home.php
- **What it provides:** Superinvestor portfolios (Buffett, Ackman, Burry, etc.), quarterly holdings updates
- **Rate limits:** Web scraping allowed (no official API)
- **Auth:** None
- **Category:** Institutional investor tracking
- **Notes:** Curated list of famous investors, historical position tracking

## 17. Insider Monkey
- **URL:** https://www.insidermonkey.com/
- **What it provides:** Hedge fund holdings, insider trading activity, stock ownership data
- **Rate limits:** Web data available (no official free API)
- **Auth:** None for web data
- **Category:** Institutional & insider data
- **Notes:** Aggregates and analyzes 13F filings, sentiment scoring

## 18. Reddit API (Official)
- **URL:** https://www.reddit.com/dev/api
- **What it provides:** Social media posts, comment sentiment, subreddit activity (r/wallstreetbets, r/stocks, etc.)
- **Rate limits:** Free tier: 60 requests/minute with OAuth
- **Auth:** OAuth (free application registration)
- **Category:** Alternative data - social sentiment
- **Notes:** Real-time retail sentiment, meme stock tracking

## 19. Twitter API v2 (Basic)
- **URL:** https://developer.twitter.com/en/docs/twitter-api
- **What it provides:** Tweets search, user timelines, real-time streams (filtered by keywords/hashtags)
- **Rate limits:** Free tier: 500,000 tweets/month (read)
- **Auth:** OAuth 2.0 (free developer account)
- **Category:** Alternative data - social sentiment
- **Notes:** Financial Twitter ($TICKER) tracking, influencer sentiment

## 20. CoinMetrics API
- **URL:** https://docs.coinmetrics.io/api/v4
- **What it provides:** Crypto network data (hashrate, active addresses, fees), market data, on-chain metrics
- **Rate limits:** Free tier: Community edition with limited metrics
- **Auth:** API key (free tier available)
- **Category:** Crypto on-chain analytics
- **Notes:** Institutional-grade crypto data, research-focused

## Implementation Notes

### Authentication Priority
Most free tier APIs require simple API key registration:
1. Register at provider website
2. Store key in QuantClaw `.credentials/` directory
3. No credit card required for free tiers

### Rate Limit Strategy
- Implement caching layer for static data (fundamentals, historical)
- Use WebSocket streams where available (Bitquery, The Graph)
- Batch requests during off-peak hours
- Combine multiple data points per request where supported

### Data Quality Considerations
- **Crypto:** The Graph > Bitquery > Covalent (by recency/reliability)
- **Macro:** OECD/BIS > Trading Economics (authoritative vs aggregated)
- **Social:** Twitter > Reddit (wider financial community coverage)
- **Stocks:** IEX Cloud > EOD Historical (real-time vs delayed)
